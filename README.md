# Educational Learning Management System

A full-stack LMS project using Next/Vite, Express, and Prisma for student and teacher management. Includes integrated Zoom Meeting SDK.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
